## 关于这个项目

> 这个项目实质是一个地图存档，在我八年级的暑假开始建造。
> 目前此项目还未完全完工。

### 1.所有者是谁？
这个项目的所有者是一名65中2020级6班的学生。

### 2.版权
本项目/存档按照 CC BY-NC-SA 4.0 协议共享。

### 3.谁参与了存档的建造？
是我和班里几个Minecraft建筑大佬朋友们一起做的。不过，后面几位大佬就不咋上线了，因此`守正楼`的教室部分是我一个人用`WorldEdit`模组搭建起来的。
